# 🥗 DietWise — AI-Powered Diet Manager

A full-featured diet management app built with React + Vite + Tailwind CSS, powered by **free AI** via Groq.

## Features
- 📊 **Dashboard** — Calorie ring, macro bars, weekly chart, water tracker
- 🍽️ **Meal Logger** — 60+ food database, custom foods, AI coach
- 🤖 **AI Recipe Finder** — Generate recipes from your ingredients (Groq AI)
- 📋 **7-Day Meal Plans** — AI-generated, personalized plans (Groq AI)
- 💪 **Exercise Tracker** — Log workouts, net calorie balance
- 👤 **Profile** — BMI, TDEE, calorie targets, weight journey
- ⚙️ **Settings** — Groq API key setup

## Getting Started

```bash
npm install
npm run dev
```

## Free AI Setup (Groq)
1. Go to [console.groq.com](https://console.groq.com)
2. Sign up (no credit card needed)
3. Create an API key
4. Add it in the app under **Settings**

## Tech Stack
- ⚛️ React 18 + Vite 5
- 🎨 Tailwind CSS 3
- 🛣️ React Router 6
- 📊 Recharts
- 🤖 Groq API (Llama 3 — free)
- 🗃️ Context API + useReducer
- 💾 localStorage persistence
